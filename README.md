sass-tumblr
===========

Ignore [Tumblr template tags](https://www.tumblr.com/docs/en/custom_themes) like ``a { color: {color:Text}; }`` while parsing SCSS.

Usage
-----

Add dependency to your Gemfile.

    gem "sass-tumblr"

Then require ``sass_tumblr`` in appropriate way.

    require 'sass_tumblr'
